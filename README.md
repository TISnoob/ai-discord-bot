# ai-discord-bot

A simple AI powered discord bot with discord.js

Required -
1. NodeJS
  - discord.js
  - openai
  - dotenv
  - djs-commander
  - replicate
2. Discord Bot (token etc)
3. OpenAI API token
4. Replicate token

Installation -
 1. rename .env.example to .env
 2. Edit .env
 3. run ```node deploy-commands.js```
 4. run ```npm i```
 5. then run ```npm start```
